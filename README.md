# Apuntes sobre *Touch Designer*

Es un lenguaje de programación visual, basado en nodos para contenido de multimedia interactivo en tiempo real. 

`¿Qué es?`

- Permite crear contenidos multimedia, arte generativo, aplicaciones 3D y experiencias interactivas en tiempo real sin necesidad de escribir código tradicional, aunque se integra perfectamente con el lenguaje Python.
